ska läsa om HTML grunden

HTML, eller Hypertext Markup Language, är språket som används för att skapa webbsidor. Det används för att strukturera innehåll på internet.

Vad är HTML?

HTML består av element som beskriver innehållet på en webbplats, inklusive text, länkar, bilder, och mycket mer.
